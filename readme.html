Welcome to the Prestige Theme for WordPress!

Prestige marks a new phase in my theme development for WordPress users and for the future with WordPress 3. Over time, theme design will get even more advanced as I continue to add more and more themes to my WordPress themes portfolio.

The Prestige theme for WordPress is an example of what is yet to come plus more. Over the span of this year, each new theme will have more features built in as I fine-tune a standard framework for my WordPress themes and templates.

Prestige Features

  • Expandable content column when the “right” sidebar is not being used; Offers full width and a Right Sidebar layout
  • HTML Snippets to give you nested 2, 3, and 4 column layouts
  • Menu system based on your own ordering of pages
  • Collapsible header when you don’t have anything published in the top area
  • 16 Widget areas – most are fully collapsible
  • Bottom Widget areas offers a different style for each group
  • Built-in Search – so you don’t have to use the search widget
  • Built-in Breadcrumbs – no plugin is needed
